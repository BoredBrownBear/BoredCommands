# BoredCommands
An updated fork of the **Fabric** mod [MyCommands](https://github.com/Markus23/mycommands) by [Lordwissend/Markus23](https://www.curseforge.com/members/lordwissend/followers). Only tested with version 1.16.5.

**Utilities**:
- /fly
- /flyspeed
- /walkspeed
- /god
- /repair
- /heal

**Teleportation**:
- /setspawn
- /spawn
- /sethome
- /delhome
- /home
- /setwarp
- /delwarp
- /warp
- /tpa
- /tpaccept, /tpyes
- /tpdeny, /tpno
- /back

**Time & Weather**
- /day
- /night
- /sun
- /rain
